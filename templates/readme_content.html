Este é um sistema de gestão de usuários desenvolvido com Flask.

Funcionalidades:

  • Cadastro e login de usuários
  • Recuperação de senha por e-mail (código de 6 dígitos)
  • Painel administrativo: listar, excluir e alterar senha de usuários
  • Tema preto e branco responsivo com sidebar colapsável
  • Relógio em tempo real
