Sobre o Projeto

📋 Sistema de Gestão de Usuários

Este é um sistema completo de gerenciamento de usuários desenvolvido como projeto de estudo e prática com Flask.

🛠️ Tecnologias Utilizadas

  • Backend: Python com Flask
  • Banco de Dados: SQLAlchemy ORM (SQLite / PostgreSQL)
  • Autenticação: Flask-Login
  • Formulários: Flask-WTF com CSRF protection
  • E-mail: Flask-Mail com SMTP do Gmail (App Password)
  • Frontend: HTML5, CSS3, JavaScript
  • Estilo: Tema Dark Responsivo
  • Deploy: PythonAnywhere / Render / Railway

✨ Funcionalidades

  • Cadastro de usuários
  • Login com sessão persistente
  • Recuperação de senha com código de 6 dígitos via e-mail
  • Painel administrativo completo
  • Listagem de usuários com busca e paginação
  • Admins podem: excluir, alterar senha e alterar permissões de outros usuários
  • Interface responsiva (funciona no celular e PC)
  • Relógio em tempo real
  • CSRF protection em todos os formulários
