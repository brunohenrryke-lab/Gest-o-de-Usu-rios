# User Management System

A production‑ready Flask web application for user management with roles, password recovery via email, and an admin panel.

## Features

- User registration, login, logout
- Password recovery with 6‑digit code sent by email (expires in 5 minutes)
- Role‑based access control (admin / regular user)
- Admin can list, delete, and change passwords of other users
- Responsive black/white theme with collapsible sidebar
- Real‑time clock on dashboard
